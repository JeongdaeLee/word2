== Description ==

Fitness Gym Club is a stunning WordPress theme with a design suited for fitness trainers, gym trainers, physical trainers, personal fitness coaches, weight centers, fitness and health consultants, aerobics classes, Zumba classes, power yoga, and other fitness-related websites. It comes with a strong Bootstrap framework making your website robust and user-friendly. This free theme is minimal and elegantly designed which will result in a responsive design that will fit every screen size and work well on every device. This comes with a sophisticated and clean layout with a beautiful design and plenty of personalization options to allow you to make some basic changes in the layout. The design comprises clean codes that are also secure enough to provide you with a safe environment for your website. These codes also follow some of the best SEO practices to fetch top ranks in searches and get more visitors coming to your web page. Call To Action Button (CTA) is going to improve your conversions and interactive part. CSS effects and stunning animations are also included along with useful social media options. Its lightweight design results in pages with a faster page load time. To provide the translation-ready capability to your website, it is provided with WPML and RTL compatibility.

For Homepage Setup:
====================

1. Go to Dashboard >> Pages >> Add New, Create a new page and name it as "Home" then select the template "Custom Home Page" and publish it.

2. Go to Dashboard >> Settings >> Reading, Select the option of Static Page, now select the page you created to be the homepage.

For Header:
============

1. Go to Dashboard >> Appearance >> Customizer >> Theme Settings >> Header, add phone number, add timing and publish it.

For Slider:
============

1. Go to Dashboard >> Pages >> Add New, Create a new Page and name it, add the content, featured image and publish the page. Repeat this step for other sliders.

2. Go to Dashboard >> Appearance >> Customizer >> Theme Settings >> Slider Settings, Click the checkbox of Show/Hide Slider, select the pages of the slider that you have created, add experience text and publish it.

For Service Section:
====================

1. Go to Dashboard >> Posts >> Category, add new category for Service section, name it and publish it.

2. Go to Dashboard >> Posts >> Add New, Create a new Post and name it, add featured image, assign the service category and publish the post. Repeat the step for other service posts.

3. Go to Customizer >> Theme Settings >> Service Section, add section title, select the category that you have created and publish it.

For Footer Section:
====================

1. Go to Dashboard >> Appearance >> Widgets, here you can add widgets to the footer widgets areas, which can be seen in footer.

For Copyright Text:
====================

1. Go to Customizer >> Theme Settings >> Footer Text, add footer text here and it also have an option to show/hide back to top.
